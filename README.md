# HNU ID2501 灵感仓库 班级工具箱 智慧平台 V2.0.0 更新公告
发布日期：2025-11-28 版本代号：Utopia

我们很高兴地宣布 ID2501 班级灵感仓库 迎来了一次重大升级！本次更新采用了全新的界面设计语言，重构了移动端体验，并引入了“智能日程”等实用功能，致力于打造更透明、高效、温馨的班级数字社区。

## ✨ 核心亮点

### 🎨 全新视觉体验
液态玻璃 UI (Glassmorphism)：全站采用现代化的毛玻璃拟态设计，通透、简洁。

动态氛围背景：引入跟随式动态光斑背景，界面不再单调，充满呼吸感。

响应式布局：完美适配桌面端大屏与移动端竖屏，视觉风格高度统一。

### 📱 移动端原生体验
App 级导航：移动端底部新增固定式导航栏，单手即可轻松切换功能。

沉浸式阅读：优化了卡片列表在手机上的显示，由多列转为单列，信息获取更直观。

首页概览：新增移动端适配的欢迎页，功能入口一目了然。

## 🚀 新增功能

### 1. 📅 智能日程看板 (Smart Timeline)
不再错过任何重要时刻！系统自动按时间顺序排列考试、作业截止及团建日期。

自动倒计时：直观展示距离每个日程的剩余天数。

过期隐藏：自动过滤已过期的事件，只关注未来。

### 2. 📢 通知与投票中心
多维度筛选：支持按“发布时间”或“重要程度”对通知进行排序。

实时搜索：输入关键词即可毫秒级检索历史通知和投票主题。

状态标识：投票卡片新增“进行中/已结束”状态标记，重要通知高亮显示。

### 3. 📚 资源仓库 Pro
快速传送门：顶部新增常用网站入口（教务系统、图钉墙、个人门户、学校官网），一键直达。

分类索引：支持按软件、网站、文档对班级资源进行分类筛选。
